A Linguagem HTML

HTML (abreviação para a expressão inglesa HyperText Markup Language, que significa Linguagem de Marcação de Texto) é uma linguagem de marcação utilizada na construção de páginas na Web. Documentos HTML podem ser interpretados por navegadores.

Alguns recursos da linguagem HTML visto até este momento:

Recurso Tags associadas Descrição
Links <a> Cria um link para outra página.
Listas <ul> e <li> Cria uma lista de itens ordenados.
<ol> e <li> Cria uma lista de itens não ordenados.
Imagens <img> Inserir uma imagem.
